## Installation

You can install the package via composer:

```bash
composer require pkboom/laravel-tinker-on-vscode
```

## Usage

```bash
php artisan tinker-on-vscode
```

You can show queries with an option:

```bash
php artisan tinker-on-vscode --query
```
